Nhập môn Web căn bản

Internet là gì?

Internet là mạng toàn cầu kết nối các thiết bị với nhau qua giao thức TCP/IP.

Web là gì?

Web là tập hợp các trang web chạy trên Internet, truy cập qua trình duyệt.

Website là gì?

Website là tập hợp nhiều trang web có cùng tên miền.

Trang web là gì?

Trang web là một tài liệu HTML hiển thị trên trình duyệt.

Browser là gì?

Browser (trình duyệt) là phần mềm dùng để xem trang web. Ví dụ: Chrome, Firefox.

Server là gì?

Server là máy chủ lưu trữ và cung cấp dữ liệu cho người dùng qua mạng.
